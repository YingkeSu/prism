🔧 查看更多可视化

使用 TensorBoard 查看完整的训练过程可视化：

tensorboard --logdir logs/visualization_demo

然后在浏览器中打开 http://localhost:6006

可用的可视化指标:

  • 标量指标: episode/reward, episode/length
  • 轨迹: trajectory/x, trajectory/y, trajectory/z
  • 动作: action/vx, action/vy, action/vz, action/omega
  • 可靠性: reliability/lidar, reliability/rgb, reliability/imu
  • 融合权重: fusion_weights/lidar, fusion_weights/rgb, fusion_weights/imu
  • 传感器: sensors/rgb (图像), sensors/lidar_*, sensors/imu_*

💡 如何使用

1. 查看视频

直接在浏览器中播放上面的视频，或在命令行中使用：

open videos/evaluation_demo-episode-0.mp4 # macOS
xdg-open videos/evaluation_demo-episode-0.mp4 # Linux
start videos/evaluation_demo-episode-0.mp4 # Windows

2. 分析轨迹数据

python3 -c " import json with open('logs/visualization_demo/train/trajectories.json', 'r') as f: data = json.load(f) print(f'Episodes: {len(data)}') for traj in data: print(f\"Episode {traj['episode']}: reward={traj['total_reward']:.2f}, steps={traj['length']}\") "

3. 训练新模型

python tests/integration/visualize_agent_environment.py --mode train --timesteps 1000

4. 评估已有模型

python tests/integration/visualize_agent_environment.py --mode evaluate --model models/trained_model.zip --n-videos 5
