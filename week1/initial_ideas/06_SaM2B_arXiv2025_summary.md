# SaM²B: Reliability-Aware Dynamic Weighting Allocation for Multi-Modal UAV Beam Prediction

**基本信息**
- **论文标题**: Empower Low-Altitude Economy: A Reliability-Aware Dynamic Weighting Allocation for Multi-Modal UAV Beam Prediction
- **作者**: (待查)
- **年份**: 2025年12月
- **arXiv ID**: 2512.24324
- **原文链接**: https://arxiv.org/html/2512.24324v1

**研究背景**
多模态UAV波束预测中，现有方法使用固定权重融合不同模态（视觉、GPS、飞行姿态、地理空间数据），但当传感器模态的可靠性因运动场景动态变化时，固定权重方法性能下降。

**核心贡献**
1. **可靠性感知的自适应权重分配**:
   - **核心创新**: 动态推断各模态的可靠性
   - 使用注意力机制和质量线索实时调整各模态贡献
   - 对比固定权重方法显著提升性能

2. **低空经济（LAE）应用场景**:
   - 针对UAV低空经济（Low-Altitude Economy）任务设计
   - 解决多模态波束预测中的可靠性问题

3. **SaM²B框架**:
   - 提出了完整的可靠性感知动态权重分配框架
   - 适用于多种UAV通信模态

**技术细节**
- **模态**: 视觉 + GPS + 飞行姿态 + 地理空间
- **机制**: 可靠性推断 + 动态权重更新
- **注意力**: 质量感知的注意力机制
- **应用**: 波束预测、通信质量评估

**实验结果**
- 在LAE场景下验证有效性
- 可靠性感知方法显著优于固定权重
- 动态权重分配快速适应传感器质量变化

**与UAV导航的相关性**
直接面向UAV通信任务，但对多模态感知融合有重要借鉴：
1. **可靠性感知**: 多模态感知中的核心问题
2. **动态权重分配**: 实时调整融合策略的关键
3. **多模态**: 视觉+IMU+GPS是UAV标准配置

**可借鉴点**
1. **传感器可靠性实时推断方法**
2. **动态权重更新机制**
3. **多模态注意力设计**
4. **质量线索提取**（信噪比、点密度、置信度等）

**局限与改进空间**
1. 主要关注通信任务，而非感知融合
2. 可靠性推断方法较简单，未利用深度学习
3. 缺乏理论保证（权重分配的收敛性、稳定性）
4. 未考虑跨模态的长期一致性

**创新机会（针对UAV感知）**
1. **深度学习可靠性感知**: 使用神经网络预测传感器质量
2. **多模态可靠性联合建模**: 考虑模态间相互影响
3. **鲁棒的权重分配**: 理论保证的动态权重优化
4. **端到端可靠性感知融合**: 将可靠性感知直接集成到融合网络中
