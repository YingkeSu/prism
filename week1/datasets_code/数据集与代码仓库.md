### 1. UAV-MM3D (ICCV 2025)

**基本信息**
- **论文**: UAV-MM3D: A Large-Scale Synthetic Benchmark for 3D Perception of Unmanned Aerial Vehicles with Multi-Modal Data
- **会议**: ICCV 2025
- **arXiv**: 2511.22404
- **链接**: https://arxiv.org/html/2511.22404v1

**数据特征**
- **模态**: LiDAR + RGB + IR (红外) + Radar + DVS (事件相机)
- **规模**: 大规模合成数据集（具体规模待查）
- **同步性**: 多模态数据在不同场景和天气条件下同步采集
- **标注**: 
  - 每帧的语义标注
  - 6-DoF位姿标注
  - 重建的3D地图
  - 新视角合成（NVS）
- **任务支持**: 语义分割、深度估计、6-DoF定位、位置识别、新视角合成

**获取方式**
- 论文未明确说明是否公开发布
- 需要联系作者或等待公开
- 基准价值: 提供标准化评估协议

**适用性**
⭐⭐⭐⭐⭐ **最适合本研究的UAV多模态感知基准**

### 2. UAVScenes (ICCV 2025)

**基本信息**
- **论文**: UAVScenes: A Multi-Modal Dataset for UAVs
- **会议**: ICCV 2025
- **arXiv**: 2512.10934
- **链接**: https://arxiv.org/html/2512.10934v1
- **GitHub**: https://github.com/sijieaaa/UAVScenes

- **模态**: 相机图像 + LiDAR点云

- **规模**: 大规模UAV场景数据集
- **标注**: 
  - 每帧语义标注（图像）
  - 每帧LiDAR语义标注
  - 准确的6-DoF位姿
  - 重建3D地图
  - 新视角合成（NVS）标注
- **任务**: 语义分割、深度估计、6-DoF定位、位置识别、新视角合成

**获取方式**
✅ **GitHub公开**: https://github.com/sijieaaa/UAVScenes
- 包含代码和数据加载脚本
- 详细的README说明使用方法

**适用性**
⭐⭐⭐⭐⭐ **完全公开，可直接下载使用**

## 📋 技术细节提取

### 从UAV-MM3D论文提取的技术细节

#### 多模态数据格式
```python
# LiDAR点云
lidar_points = np.from_file('lidar_points.bin')  # [N, 3]
lidar_semantic = np.from_file('lidar_semantic.bin')  # [N] 语义标签

# RGB图像
rgb_image = cv2.imread('rgb_image.png')  # [H, W, 3]

# IR图像
ir_image = cv2.imread('ir_image.png')  # [H, W, 3]

# 6-DoF位姿
pose_6dof = np.from_file('pose.bin')  # [x, y, z, qw, qx, qy, qz, tx, ty, tz]
```

#### LGFusionNet架构（来自FusedVisionNet摘要）
```python
class CrossAttentionFusion(nn.Module):
    def __init__(self, rgb_dim=256, lidar_dim=256, depth_dim=128):
        super().__init__()
        
        # 编码器
        self.rgb_encoder = ResNetEncoder(rgb_dim)
        self.lidar_encoder = PointNetEncoder(lidar_dim)
        
        # 深度引导跨注意力
        self.depth_attn = nn.Linear(depth_dim, 1)
        
        # 跨模态融合
        self.fusion = nn.Sequential(
            nn.Linear(rgb_dim + lidar_dim, 512),
            nn.ReLU(),
            nn.Linear(512, 256)  # 最终特征维度
        )
```

## 💡 创新点技术支撑

### 数据集可用性对创新点的影响

| 创新方向 | 需要的数据集 | 可用性 | 建议 |
|-----------|------------|--------|--------|
| 可靠性感知融合 | 多模态同步数据 | ⭐⭐⭐⭐ | 可用，优先使用UAVScenes |
| 跨模态Transformer融合 | 不同模态数据 | ⭐⭐⭐ | 可用UAVScenes或自定义合成 |
| 运动感知时序融合 | 运动标注数据 | ⭐⭐ | 需要额外标注，可扩展SynDrone |
| 深度引导融合 | 带深度标注的数据 | ⭐⭐⭐ | UAVScenes有，TIERS有LiDAR点云 |
