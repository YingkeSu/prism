# LVCP: LiDAR-Vision Tightly Coupled Collaborative Real-Time Relative Positioning

**基本信息**
- **论文标题**: LVCP: LiDAR-Vision Tightly Coupled Collaborative Real-Time Relative Positioning
- **作者**: (待查)
- **年份**: 2024年7月
- **arXiv ID**: 2407.10782
- **原文链接**: https://arxiv.org/html/2407.10782v1

**研究背景**
在GPS拒止环境下，UAV与UGV（无人地面车辆）的相对定位是核心挑战。如何有效融合UAV单目相机和UGV LiDAR实现鲁棒、实时的相对定位。

**核心贡献**
1. **紧耦合LiDAR-视觉相对定位**:
   - **核心创新**: 紧耦合融合相机和LiDAR进行相对定位
   - 不依赖GPS，纯视觉+LiDAR方案
   - 实时性满足协作需求

2. **粗到细两阶段框架**:
   - 粗定位：快速估计相对位姿
   - 细定位：粒子群优化精化
   - 提升定位精度和速度

3. **空地协作**:
   - UAV与UGV的相对位姿估计
   - 适用于配送、跟随、编队等任务
   - GPS拒止环境下的解决方案

**技术细节**
- **传感器**: UAV单目相机 + UGV LiDAR
- **融合**: 紧耦合融合
- **优化**: 粗到细两阶段
- **方法**: 粒子群优化

**实验结果**
- 在GPS拒止场景下验证
- 实时性满足需求
- 定位精度显著提升

**与UAV导航的相关性**
直接面向UAV空地协作：
1. **无GPS定位**: 核心挑战和解决方案
2. **多模态相对定位**: 视觉+LiDAR融合
3. **实时空地协作**: 编队、配送等应用

**可借鉴点**
1. **紧耦合融合策略**: 避免延迟累积
2. **粗到细优化**: 平衡速度和精度
3. **多模态相对定位**: 无GPS环境的关键技术
4. **协作定位框架**: 相对位姿估计

**局限与改进空间**
1. 主要关注相对定位，未考虑绝对定位和建图
2. 单目相机对深度的依赖
3. 固定的融合权重（未动态调整）
4. 对动态场景适应性有限

**创新机会（针对UAV）**
1. **多UAV绝对定位**: 扩展到多UAV场景
2. **动态权重分配**: 根据传感器质量调整
3. **预测性融合**: 预测相对运动
4. **轻量化相对定位**: 针对嵌入式平台设计
